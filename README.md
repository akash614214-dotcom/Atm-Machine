# ATM Management System (C Language)

A console-based ATM Management System developed in C language. This project simulates basic banking operations such as account creation, login authentication, balance inquiry, deposits, withdrawals, and customer management.

## Features

- Account Number + PIN Authentication
- PIN Protection with 3 Login Attempts
- Open New Account
- Account Login
- Check Balance
- Deposit Money
- Withdraw Money
- View Customer Information
- Logout System
- Exit Program
- List All Customers (Admin/Personal Use)

## Concepts Used

- Functions
- Structures
- Arrays
- Loops
- Conditional Statements
- User Authentication
- Menu-Driven Programming
- Runtime Memory Management
- Data Searching

## Project Flow

1. Create a New Account
2. Login using Account Number and PIN
3. Access ATM Services:
   - Check Balance
   - Deposit Money
   - Withdraw Money
   - View Account Information
4. Logout
5. Exit Program

## Security Features

- Account Number + PIN Verification
- Maximum 3 Login Attempts
- Unauthorized Access Prevention
- User-Specific Account Access
